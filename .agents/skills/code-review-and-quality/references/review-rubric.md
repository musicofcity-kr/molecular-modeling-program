# Review Rubric

Score each area 0-2.

| Area | 0 | 1 | 2 |
|---|---|---|---|
| Chemistry validation | absent | partial | gated and tested |
| UI clarity | confusing | usable | classroom-ready |
| Type safety | weak | adequate | strong |
| Test coverage | absent | partial | valid+invalid covered |
| Dependency discipline | undocumented | partial | logged and justified |

A change should not be accepted below 7/10 for MVP core features.
