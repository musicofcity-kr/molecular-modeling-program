# Classroom Language Bank

## Status

- 검증 완료
- 검토 필요
- 구조를 먼저 그려주세요
- 계산을 위해 구조 검증이 필요합니다
- 현재 구조는 분자 구조로 해석되지 않았습니다

## Explanation

- 분자식은 분자에 포함된 원자의 종류와 개수를 나타냅니다.
- 분자량은 구성 원자의 상대 원자량을 합한 값입니다.
- 3D 구조는 분자의 입체적 배치를 이해하기 위한 모델입니다.

## Misconception Prevention

- 구조식의 선 하나는 보통 공유 전자쌍 하나를 나타냅니다.
- 2D 구조식이 실제 분자의 모든 입체 모양을 그대로 보여주는 것은 아닙니다.
- 생성된 3D 모델은 조건과 계산 방법에 따라 달라질 수 있습니다.
